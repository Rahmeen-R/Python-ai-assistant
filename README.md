AI‑powered interactive Python learning tool built with Streamlit, LangChain, and Gemini.
This project demonstrates my ability to build real-world AI applications with clean architecture, modern LLM integration, and a user‑focused interface. The Python AI Assistant provides an interactive chat experience where users can explore Python concepts through clear explanations generated by Gemini 2.5 Flash.

How to Run:

Add your Gemini API key inside a .env file using GEMINI_API_KEY=your_key. The app loads this key automatically at startup. After installing the required packages from requirements.txt, start the assistant by running streamlit run app.py, which opens the Python AI tutor in your browser.
